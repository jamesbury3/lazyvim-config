# Neovim Config

## Setup

1. Follow the [LazyVim installation docs](https://www.lazyvim.org/installation).
2. Clone this config to `~/.config/nvim`.
3. Add the utilities to your shell profile.

For Bash, add this to `~/.bashrc`:

```bash
source ~/.config/nvim/nvim-utils.sh
```

For PowerShell on Windows, add this to `$PROFILE`:

```powershell
. "$HOME/.config/nvim/nvim-utils.ps1"
```
